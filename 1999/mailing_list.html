Email Instructions For the HPDP Mailing List

Click on any underlined link to send an email for that command
(this is dependent on your email program being configured correctly)

hpdp-subscribe@cybercom.com.au
 Allows the user to Subscribe to the mailing list.The user will be sent a Confirmation Message which they must Reply To. This confirms that the user REALLY wants to join the List, and the list is not being hi-jacked by another Internet User. Once Confirmed, the user will receive the Welcome Message. All subscribe requests have to be approved first by the list administrator.
hpdp-remove@cybercom.com.au
 To Unsubscribe, the user sends an email to this address.
hpdp@cybercom.com.au
 Once a user is a member, (s)he can send email to this address.
Their message will be sent to all members of the mailing list.
hpdp-index@cybercom.com.au
 Returns a list of subjects of the last 100 messages sent to the list.
Can be useful if you have been "off the air" for a while.
hpdp-index.55_150@cybercom.com.au Returns a specific range of message subjects (e.g. 55 to 150).
hpdp-get@cybercom.com.au
 Returns the full text of the last 10 messages.
hpdp-get.77@cybercom.com.au
 Returns the full text of a specific message (e.g. 77).
petert@cybercom.com.au
 Send Peter T. an email if you have any difficulties.
