All,

We need to make a fairly urgent decision on the issue of non-standard pool comps. It is very possible that this Wed night could leave us in the same situation as last week should the attendee numbers be that low again.

So here are what I see as the basic issues that we need to address. Please consider these and contribute to the online discussion (forget work, it's a Monday after all). If these questions are agreeable to all, we will post a poll on the website with the view to resolving this prior to Wed night.

Unlike with John L's favourite namesake, Little Johnny, we shan't be confusing the referendum question by tying complex conditions to a fairly basic question and therefore guaranteeing the result.

Submit your opinion NOW
Read more details on the surveyNON-STANDARD POOL COMP REFERENDUM

1. Should we allow 16-player comps at all? Yes/No

2. Should we allow 24-player comps at all? Yes/No

3. Should ANY non-standard game (i.e. not 32 players) qualify for points distribution? Yes/No

4. If we do allow non-standard comps to earn points, should we:
a. allocate full points (i.e. 16 pts for 1st, etc),
b1. allocate half points (i.e. 8 pts for 1st, etc) - 16-player comp
b2. allocate 75% points (i.e. 12 pts for 1st, etc) - 24-player comp, or
c. use some other form of scoring which may be more fair?

And another question remains which may be more difficult to decide upon. It may require that we use a one-off solution this Wed should we find ourselves in this situation.

5. If we allow 24-player comps, how do we decide the 2 finalists? Suggestions requested.

I also think it fair that whatever is decided here is retrospectively applied to last week's game. The only other option for last week is to use the existing precedent which means that, as a non-standard comp, no points will be awarded.
